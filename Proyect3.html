PORTAFOLIO

AccesibilidadFit

Rol en el proyecto

Como diseñador de UX, mi rol central consistió en liderar la creación de una interfaz de usuario accesible y eficiente para la aplicación de asesoramiento en actividad física para personas no videntes.

Objetivo del proyecto

Desarrollar una aplicación inclusiva que brinde asesoramiento personalizado en actividad física, fomentando un estilo de vida saludable para la comunidad no vidente.

Publico objetivo

Individuos no videntes de diversas edades y niveles de habilidad física, buscando orientación para realizar ejercicios adaptados a sus necesidades específicas.

Principales desafios

Afrontar la falta de herramientas accesibles para la actividad física, garantizando que la aplicación fuera intuitiva y motivadora para una audiencia no vidente.

Detalles del estudio de investigación

Se llevó a cabo una investigación exhaustiva que incluyó entrevistas y pruebas de usabilidad con usuarios no videntes. El estudio se centró en comprender las necesidades específicas, los desafíos y las preferencias de esta audiencia para informar el diseño.

Resultados de pruebas de usuario

Las pruebas de usuario destacaron la eficacia de la navegación por voz, la claridad de las descripciones auditivas y la importancia de la retroalimentación continua para mejorar la experiencia.

Conclusión

El proyecto logró crear una aplicación de asesoramiento en actividad física que no solo abordó los desafíos de accesibilidad, sino que también proporcionó una experiencia enriquecedora y motivadora para la comunidad no vidente. La retroalimentación continua de los usuarios fue fundamental para las mejoras iterativas, asegurando que la aplicación evolucionara para satisfacer las necesidades cambiantes de los usuarios no videntes.
